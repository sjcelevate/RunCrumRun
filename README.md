# Run Crum Run v0.4

This version is built from the fuller v0.2 gameplay, not the simplified v0.3 test file.

## Kid-focused additions
- Rocket boost audio when Colton hits water
- Sustained rocket sound during boost
- Bigger jump sound and splash ring
- Star combo system
- Mystery boxes
- Star Magnet power-up
- Giant Colton power-up
- Smash obstacles while giant
- More text pops: ROCKET BOOST, COMBO, SMASH, BONK
- Better confetti and win effects
- More Colton branding in the city

## Controls
- Phone/tablet: swipe left/right, tap or swipe up to jump
- Keyboard: arrows or A/D, Space/Up/W to jump
